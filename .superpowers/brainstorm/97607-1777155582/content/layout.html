A — Continuous Scroll

One long page, scroll down through all sections. Dot navigation on the side shows position. Smooth, natural flow. Most common portfolio style — works well with parallax animations.

SNAP-SCROLL — EACH SECTION = FULL SCREEN

B — Snap-Scroll Sections

Each section snaps to fill the full screen — like a game menu or starmap. Scroll snaps to the next "world." More immersive, dramatic. Pairs perfectly with SC2-style entry animations per section.

C — Starmap Navigation

An SC2-style galaxy map as the main navigation — click nodes to jump to sections. Ultra creative, totally unique. High effort, high reward. Makes the portfolio itself feel like an interactive experience.
