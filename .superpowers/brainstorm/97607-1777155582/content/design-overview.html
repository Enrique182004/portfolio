05 // Implementation
TECH STACK

Framework: Next.js 14 (App Router) + TypeScript

Styling: Tailwind CSS + CSS custom properties for the design tokens

Animations: GSAP (panel transitions, node entry) + CSS animations (particles, hex pulses)

Starmap: HTML5 Canvas 2D — hex grid drawn procedurally, particles managed in a render loop

GitHub Data: GitHub REST API v3 — fetched server-side in Next.js, cached with ISR (revalidate every hour)

Fonts: Space Mono (Google Fonts) + Inter

Deployment: Vercel (auto-deploys from GitHub main branch)

GitHub Pages: The portfolio itself can be hosted on GitHub Pages via gh-pages — making it your github.io page

Demo Previews: iframe modal for repos with live URLs, sandboxed

No DB needed — all data from GitHub API + static content
