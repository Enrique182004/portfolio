01 // Node-to-Node Transition Flow
1

HOVER NODE

Node ring expands 10%, connection line brightens, small orbiting particle appears around the node, cursor changes to pointer.

2

CLICK — ENERGY BURST

Node flashes gold, a radial burst ripple expands outward from it, connection lines all briefly glow then fade. Duration: 300ms.

3

MAP DIMS + PANEL RISES

Starmap fades to 20% opacity. Panel slides up from bottom with a slight overshoot (spring easing). Background hex grid stays faintly visible behind it.

4

CONTENT STAGGERS IN

Panel header appears first, then content items cascade in one by one (60ms stagger). Feels like a HUD loading data.

5

NODE-TO-NODE (WHILE PANEL OPEN)

Old panel slides out (left or right based on node position). Map briefly brightens, camera "drifts" toward new node, new panel slides in. No full close needed — seamless jump.

02 // Always-On Ambient Animations
✦

NODE BREATHING

All nodes slowly pulse their glow on a 3–4s sine cycle, each slightly offset so they never pulse in sync — feels alive, not mechanical.

⟿

FLOWING CONNECTIONS

Small bright dots travel along the dashed lines from center to each node, like data signals. One dot per line, looping at different speeds.

⬡

HEX GRID MOUSE REACTION

Hexagons near the cursor glow slightly brighter. The effect radius follows your mouse — like pressing a light through fabric.

·

PARTICLE DRIFT

~120 particles float slowly. Each has its own speed, direction, and opacity cycle. No two move the same way.

↗

CURSOR TRAIL

A 6–8 dot trail follows the cursor on the map — each dot fades in size and opacity. Subtle but adds depth to interactions.

◎

PAGE LOAD BOOT

On first visit: screen flickers on, nodes appear one by one, connection lines draw themselves outward from center. Total: ~2s intro sequence.

Clicking ▶ DEMO opens a full-screen modal with an iframe showing the live project. Header shows repo name + live URL. "Open in new tab" button. Glassmorphism-style backdrop blur.

SKILL TREE HOVER

Hovering a skill leaf shows a tooltip: skill name + "Used in: [project names]". The branch above the hovered skill also brightens, tracing the path up to root.
